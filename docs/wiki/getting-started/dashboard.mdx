## Επισκόπηση

Στο Dashboard βλέπεις:
- Συντομεύσεις προς τα τμήματα που χρησιμοποιείς πιο συχνά.
- Ειδοποιήσεις (π.χ. χρήστες σε αναμονή έγκρισης, νέα imports).
- Πρόσφατη δραστηριότητα.

## Πλοήγηση

- **Sidebar αριστερά:** όλα τα τμήματα στα οποία έχεις πρόσβαση.
- **Topbar:** αναζήτηση, ειδοποιήσεις, βοήθεια (εικονίδιο `?`), προφίλ.
- **Mobile:** κάτω εμφανίζεται bottom bar με τις βασικές συντομεύσεις.
