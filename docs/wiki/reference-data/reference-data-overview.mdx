## Τι περιλαμβάνεται

- Νομοί, Δήμοι, Περιφέρειες (ΓΕΜΗ Open Data)
- Κατηγορίες ΦΠΑ
- Λοιπά lookup tables

Για κάθε πίνακα φαίνεται πλήθος εγγραφών, ημερομηνία τελευταίας ενημέρωσης και πηγή.

## Χειροκίνητη ανανέωση ΓΕΜΗ

Όποιος χρήστης έχει δικαίωμα `metadata.manage` βλέπει κουμπί <strong>Sync</strong> δίπλα σε πίνακες ΓΕΜΗ.

<Callout type="warning">
  Το sync τραβάει νέα δεδομένα από Open Data και επικαλύπτει τοπικές αλλαγές.
</Callout>
